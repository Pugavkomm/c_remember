# c_remember
## Реализация разных сортировок и структур данных на с++ 

### Сортировки
**Сортировка пузырьком:**
```
bubble_sort.cpp 
```
**Пирамидальная сортировка**
```
heap_sort.cpp
```

### Структуры данных 
**Одно-связный список**
```
list.cpp
```
**Бинарное дерево поиска**
```
tree.cpp
```
Все файлы сортировок находятся в sort.
